
# pnet网络

如果可以的话,节点与节点之间,可以任意互相连接,每个节点它只维护与它相邻节点的连接信息。所以你可以无限的扩展，无限的连接更多的节点。但是通常不建议你那么做，因为这样会使得消息经过的节点太多。
